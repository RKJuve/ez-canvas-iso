ez-canvas-iso
=============

first attempts at a native canvas isomorphic display engine
![v0.0.0](/etc/v0.0.0.png)

1. ```git clone https://github.com/RKJuve/ez-canvas-iso.git && cd ./ez-canvas-iso```
2. ```npm install```
3. ```npm start```
4. open browser to http://localhost:3000
